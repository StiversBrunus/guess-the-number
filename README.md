# 🎯 Guess the Number

Projeto simples de um jogo de adivinhação desenvolvido com **HTML, CSS e JavaScript puro**, com foco em prática de **lógica de programação**, manipulação de DOM e estruturação de código.

---

## 📌 Objetivo

Este projeto foi criado para reforçar conceitos fundamentais como:

- Entrada de dados do usuário
- Estruturas condicionais (`if/else`)
- Manipulação de elementos HTML (DOM)
- Organização e reutilização de funções
- Controle de estado (tentativas, número secreto)

---

## 🕹️ Como funciona

O jogo gera um número aleatório entre **1 e 10**.

O usuário deve tentar adivinhar esse número.

A cada tentativa, o sistema informa se o número secreto é:

- 🔼 Maior
- 🔽 Menor

Quando o usuário acerta:

- O jogo exibe uma mensagem de sucesso
- Mostra a quantidade de tentativas
- Libera o botão de reiniciar

---

## 🧠 Conceitos aplicados

- `querySelector` → seleção de elementos no DOM  
- `innerHTML` → alteração de conteúdo HTML  
- Funções reutilizáveis  
- Operador ternário  
- Geração de números aleatórios (`Math.random`)  
- Limpeza de input  
- Controle de fluxo
